# Expense Receipt Submission - Standard Operating Procedure (SOP)

This document outlines the standard operating procedure for submitting expense receipts using the HOM Mobile Assistant app.

## 1. Overview
The Expense Receipt component allows House of Mercy staff to submit expense records along with photographic or PDF evidence of receipts directly from their mobile devices. The app connects to Quickbase to store this information and captures necessary data for accounting purposes.

## 2. Accessing the Expense Receipt Form
- Log in to the HOM Mobile Assistant application.
- Navigate to the Expense Receipt section from the main dashboard or menu.

## 3. Completing the Expense Form
The form requires several key pieces of information. All required fields must be completed before submission.

### A. Vendor Selection
1. Tap the "Vendor" field.
2. Begin typing the name of the vendor (minimum 2 characters).
3. Select the correct vendor from the dropdown list that appears.
4. If you select the wrong vendor, use the clear button to reset the selection.

### B. Expense Details
1. **Expense Date**: Enter the date the expense was incurred.
2. **Quantity (Qty)**: Enter the number of items purchased (default is 1).
3. **Amount ($)**: Enter the total amount of the expense.

### C. Grant Information
If the expense is related to a specific grant, you must indicate this:
1. Under "Grant Expense?", select **Yes**.
2. A secondary list of grant types will appear. Select the appropriate grant:
   - Reentry Grant
   - Federal Grant
   - Tax Grant
   - Coalition Grant (EEC)
3. If it is not a grant expense, leave the selection as **No**.

### D. Description
Enter a detailed description of the item(s) purchased and/or the reason for the purchase. If a grant is selected, the system will automatically append the grant type to your description upon submission.

## 4. Attaching the Receipt
You must attach evidence of the receipt. You can either take a new photo using your device's native camera or upload an existing file (Image or PDF).

1. Tap the **Capture Receipt** button.
2. You will be prompted to either take a photo using your device's camera app or select an existing file/PDF from your device.
3. If you take a photo, review the captured image. If it is unclear, tap **Remove File** and try again.
4. If you upload a PDF, a document icon will appear to confirm the attachment.

*Note: The system automatically renames the file upon submission to include the date, amount, and your name (e.g., `MM-YY $##.00 StaffName.png` or `.pdf`) to meet auditing standards.*
